# Introduction

Let's discover **use-midi in less than 5 minutes**.

## Installation

To install the entire `use-midi` lib:

```shell
yarn add use-midi
```

## Getting Started

```shell
git clone https://github.com/tseijp/use-midi
```

Run the development server:

```shell
cd use-midi
yarn init
yarn docs
```

Your site starts at [http://localhost:3000](http://localhost:3000).
